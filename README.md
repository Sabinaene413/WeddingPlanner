# WeddingPlanner API

A simple REST API for managing weddings and wedding-related tasks.

This project is built as a learning project to practice:
- ASP.NET Core Web API
- REST principles
- DTOs and validation
- Clean controller structure

---

## 🛠️ Tech Stack

- .NET 9
- ASP.NET Core Web API
- In-memory data storage (for now)
- Swagger / OpenAPI

---

## 📌 Features

- CRUD operations for Wedding Tasks
- DTO-based request/response models
